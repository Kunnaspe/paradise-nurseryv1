# Paradise Nursery

Paradise Nursery is a React + Redux shopping experience for houseplants. Users can browse plants by category, add items to a cart, update quantities, remove items, and view totals.

## Features
- Landing page with company name and “Get Started” button
- Product listing page:
  - Plants grouped into 3+ categories
  - 6+ unique plants per category
  - Add to Cart disables after adding
  - Navbar with Home / Plants / Cart and dynamic cart count
- Cart page:
  - Shows item totals and cart total
  - Increase/decrease quantity
  - Delete item
  - Checkout button shows “Coming Soon”
  - Continue shopping link
